# MediCare Assistant - Personal Health Companion 🏥

A comprehensive medical assistant web application that helps users manage their health, track symptoms, medications, and appointments. Built with modern web technologies and featuring a beautiful, responsive design.

## ✨ Features

### 🏠 Dashboard
- **Health Score Tracker** - Dynamic health score calculation based on various health metrics
- **Quick Actions** - Fast access to key features like symptom checker, BMI calculator, and appointment scheduling
- **Recent Activity** - Timeline of recent health activities and medication intake

### 🩺 Symptom Checker
- **Interactive Symptom Selection** - Choose from common symptom categories or describe symptoms freely
- **AI-Powered Analysis** - Intelligent symptom analysis with recommendations and urgency assessment
- **Medical Advice** - When to see a doctor recommendations based on symptoms
- **Direct Appointment Booking** - Quick scheduling from symptom results

### 📊 Health Metrics
- **BMI Calculator** - Calculate Body Mass Index with category classification and health advice
- **Vital Signs Tracker** - Record blood pressure, heart rate, temperature, and oxygen saturation
- **Health History** - Comprehensive tracking of all health metrics over time
- **Visual Feedback** - Color-coded health indicators and progress tracking

### 💊 Medication Management
- **Smart Medication Tracking** - Add medications with dosage, frequency, and timing
- **Automatic Reminders** - Real-time medication reminder notifications
- **Compliance Monitoring** - Track medication adherence and impact on health score
- **Easy Management** - Edit, delete, and mark medications as taken

### 📅 Appointment Scheduling
- **Comprehensive Booking** - Schedule appointments with doctor details, type, and notes
- **Appointment Types** - Support for checkups, consultations, follow-ups, emergencies, and specialist visits
- **Smart Validation** - Prevents scheduling appointments in the past
- **Appointment History** - View upcoming and past appointments

### 🔒 Data Persistence
- **Local Storage** - All data saved locally in your browser
- **Privacy-First** - No data sent to external servers
- **Instant Sync** - Real-time updates across all features
- **Data Export Ready** - Easy to extend for data backup features

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software installation required

### Installation
1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start using the medical assistant immediately!

### Usage
1. **Dashboard**: Get an overview of your health status and quick access to all features
2. **Track Symptoms**: Use the symptom checker to analyze symptoms and get recommendations
3. **Monitor Health**: Calculate BMI and record vital signs regularly
4. **Manage Medications**: Add your medications and receive timely reminders
5. **Schedule Appointments**: Book and manage medical appointments
